AP Computer Science Principles

Room 263G

Advanced Placement Computer Science Principles utilizes the code.org platform to guide students through learning computer programming. Students in this class will learn and apply the Javascript programming language, 98% of websites on the internet utilize Javascript (JS) to create a functioning website. Javascript along with HTML and CSS are the core technologies of the internet, also known as the World Wide Web. In AP CSP students will explore what the internet is and how it works, digital safety, binary notation, cybersecurity and other contextual information to put Computer Science in Context. In Javascript students will learn about variables, loops, traversals, conditionals, functions, lists, dictionaries, algorithms, parameter, return, libraries and app design. Students in AP CSP will complete a traditional AP examination, however students in this class will also complete a coding project that will be submitted in addition to the examination.
